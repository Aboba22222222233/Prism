## 📱 Экосистема Prism

Проект состоит из двух связанных приложений:

1.  **Web Платформа** (Учителя и Ученики)
    *   Дашборд учителя: аналитика класса, риски, управление заданиями.
    *   Дашборд ученика: дневник, статистика, чат с ментором.
2.  **Мобильное Приложение** (Только Ученики)
    *   Быстрые чек-ины (настроение, сон, энергия).
    *   AI-поддержка в кармане.
    *   Push-уведомления (планируется).

## 🚀 Возможности

### Для учеников (Web & Mobile)
| Функция | Описание |
|---------|----------|
| **Ежедневный чек-ин** | Запись настроения (1-5), энергии, сна и факторов стресса |
| **AI Ментор (Клаудик)** | Персональный чат-бот для поддержки и советов |
| **Дневник** | История всех записей с календарем и графиками |
| **Задания** | Просмотр и выполнение заданий от учителя |
| **Ресурсы** | Библиотека техник самопомощи (дыхание, медитации) |

### Для учителей (Web)
| Функция | Описание |
|---------|----------|
| **Дашборд класса** | Общая статистика, тепловая карта настроения, список учеников |
| **AI Анализ риска** | Автоматическое выявление учеников в группе риска (0-10) |
| **Профили учеников** | Детальная история и AI-сводка по каждому ученику |
| **AI Ассистент** | Помощник для анализа динамики класса |
| **Управление** | Создание классов, заданий и объявлений |

## 🛠 Технологический стек

### Frontend (Web)
*   **React 18** + **Vite**
*   **TailwindCSS** (стилизация)
*   **Recharts** (графики)
*   **Framer Motion** (анимации)

### Mobile (iOS/Android)
*   **React Native** + **Expo**
*   **Expo Router** (навигация)
*   **NativeWind** (Tailwind для Mobile)

### Backend & AI
*   **Supabase** (Auth, Database, Realtime)
*   **Groq API** (Сверхбыстрая генерация текста)
*   **Модели:** `openai/gpt-oss-120b` (основная), `llama-3.3-70b` (резерв)

## 📦 Установка и Запуск

### Предварительные требования
*   Node.js 18+
*   Аккаунт Supabase
*   Ключ Groq API (для AI)

### 1. Клонирование
```bash
git clone https://github.com/Aboba22222222233/Prism.git
cd Prism
```

### 2. Запуск Веб-версии
```bash
npm install
npm run dev
# Открыть http://localhost:5173
```

### 3. Запуск Мобильного приложения
```bash
cd prism-mobile
npm install
npx expo start
# Сканировать QR код через Expo Go (Android/iOS)
```

## 🔐 Безопасность (Demo Mode)

В данной ветке (`main`) проект настроен для публичной демонстрации:
*   🔑 **API Ключи:** Жестко прописаны в коде для стабильности показа.
*   🌍 **AI Запросы:** Идут напрямую с клиента на Groq (без прокси).
*   💾 **Кэширование:** Отключено для демонстрации "живых" ответов.

**Для продакшена необходимо:**
1.  Перенести ключи в `.env`.
2.  Включить Supabase Edge Functions.
3.  Настроить RLS политики.
